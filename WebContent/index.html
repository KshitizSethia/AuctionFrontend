Auction Game

Instructions for Auction

  • This is a closed auction, you can't see the other player's bids. You will only see the winning bid of previous rounds.
  • Winning:
    • ROUND: Bid the highest amount (float) for the painting. In case of tie, first highest bidder wins.
    • GAME: Get 'n' copies of the same painting before anyone else does. 'n' can be set in "Winning Score" on the right.
  • Controls (on the right):
    • "Enter" to add player and place bid.
    • "Ctrl + Enter" to start Auction
  • Add players, start auction! Godspeed!
